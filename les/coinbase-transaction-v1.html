Terminology

transactionId

The transaction id of the transaction.

transactionDate

The date / dateTime of when the transaction took place.

transactionAmount

Amount to be transfered with the transaction.

currencyCode

Type of currency that was transfered. E.g. USDC.

sourceAddress

The wallet address of the sender.

depositAddress

The wallet address of the recipient.
